# 校园网自动登录工具

## 简介

本项目是一个用于 Windows 下**自动检测校园网连接状态并自动登录**的现代化桌面工具。界面美观，操作简便，支持**开机自启**、**系统托盘**、**实时日志**等功能。适配以深澜（Srun）为代表的主流校园网认证系统，自动检测掉线并重连，极大提升校园网体验。

---

## 功能特性

- **自动检测网络状态**：后台定时检测网络连通性，断线自动登录。
- **手动登录/保存配置**：可随时手动发起登录，配置存储本地，重启自动读取。
- **开机自启（双保险）**：注册表 + 启动目录双重自启方案。
- **系统托盘驻留**：关闭窗口后驻留托盘，双击可恢复主界面。
- **实时流式日志**：日志动态显示，可一键清空/保存。
- **主题自适应**：支持跟随系统深色/浅色主题。
- **账号密码本地保存**：仅本地存储，不上传云端。

## 文件结构

```
SZU_network/
```

## 快速开始

### 1. 环境准备

- **操作系统**：仅支持 Windows
- **Python 版本**：>= 3.8
- **依赖库**：

  ```bash
  pip install customtkinter pystray requests pillow
  ```

  > 注：Python3.11+ 建议使用官方 Python 安装包（自带 Tcl/Tk），部分精简版/虚拟环境可能缺少 GUI 支持。

### 2. 运行程序

```bash
python main.py
```

首次运行会自动生成`srun_config.json`配置文件。

## 使用说明

### 1. 填写登录信息

- **用户名**：通常为你的学号（或带后缀如`@hlw`）。
- **密码**：校园网认证密码。
- **开机自启**：勾选后，系统下次启动会自动运行本工具。

### 2. 功能按钮

- **保存配置**：保存输入的用户名、密码、自启设置。
- **手动登录**：立即发起一次登录流程（建议首次使用时手动登录，测试配置是否正确）。
- **日志面板**：实时显示网络状态、自动登录、异常信息，可一键清空/保存。

### 3. 系统托盘

- 关闭窗口后，程序不会退出，而是最小化至系统托盘。
- 托盘右键菜单支持**恢复主界面**、**手动登录**、**退出程序**。

## 高级说明

### 配置文件说明（srun_config.json）

```json
{
  "username": "你的用户名",
  "password": "你的密码",
  "auto_start": true,
  "check_interval": 60,
  "theme": "system"
}
```

- `auto_start`：开机自启
- `check_interval`：网络检测间隔（秒）
- `theme`：主题（system/Light/Dark）

### 登录协议与安全

- 登录加密完全仿照官方 JS，采用自定义 Base64 与 XXTEA 变体算法，兼容深澜 Srun 4.x/5.x 协议。
- **本地保存密码**：配置文件明文保存密码，请妥善保护本地环境。
- **不会上传账号信息到任何第三方服务器**。

## 常见问题

#### 1. 登录失败怎么办？

- 检查用户名、密码是否填写正确。
- 若提示“无法获取ac_id或ip”，请确认已连接校园网 SSID。
- 若提示“开机自启动设置失败”，尝试**以管理员身份运行**程序。

#### 2. 日志/界面乱码？

- 请使用简体中文系统，或确保 Python 控制台/文件编码为 UTF-8。

#### 3. 如何卸载/取消自启？

- 在主界面取消勾选“开机自启”并保存。
- 删除项目目录即可（会自动清理注册表与启动项脚本）。

## 模块说明

### main.py

- 图形界面、配置管理、日志、系统托盘、核心控制。

- 网络状态检测线程，掉线自动回调登录。
